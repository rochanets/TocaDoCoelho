# Arquivos genéricos AutoToca

Insira manualmente os PDFs padrão nesta pasta com os nomes abaixo:

- `generic_contrato_original.pdf`
- `generic_minuta_cliente.pdf`

Esses arquivos são usados quando:

- Campo 8 estiver como **não se aplica** (`GENERIC_CONTRATO_ORIGINAL_FILE`)
- Campo 9 estiver como **Não** (`GENERIC_MINUTA_CLIENTE_FILE`)
